## 🎓 关键经验（下次复用）

### 1. "主从并行" 模式
- 派 2-3 Agent 是甜点区间
- 太多（5+）成本翻倍但提速有限
- 每个 Agent 独立 context，Token 翻倍

### 2. 规则引擎优先于 LLM
- Layer 1 规则 12/12 够用
- Layer 2 LLM 接入成本高、价值待验证
- 启发式已经覆盖 80% 场景

### 3. Auto mode 安全设计
- AI 不能自己加权限（防自我提权）
- AI 不能改自己的 settings（stage 2 拦截）
- 必须**用户手动**改配置
- 文档化比自动化重要

### 4. 并行 ≠ 多模型
- 用户最初目标"实现视频同款" = 多模型大乱斗
- 实际需求 = 并行加速（**不需要多模型**）
- 重新对齐目标后，少走 80% 弯路

### 5. 测试驱动规则调整
- 12 个测试用例保护规则质量
- 加新规则前先跑测试
- 改后跑测试确认不退化
