## 故意没加的（请保持弹窗）

| 规则 | 原因 |
|:-----|:-----|
| `Bash(git push *)` | 推远程是敏感操作，让你看一眼 |
| `Bash(rm -rf *)` | 危险 |
| `Bash(format *)` | 危险 |
| `Edit(C:/...)` | 不让 AI 改 C 盘 |
| `Read(//c/Windows/**)` | 不让 AI 读系统文件 |
| `Bash(*)` | 完全关安全 |

## 如果你想再激进一点

可以加（按风险从低到高）：

```json
"Bash(git push *)",                          // 推远程（让你 diff 完再确认）
"Bash(curl *)",                              // 任意 HTTP 请求
"Bash(powershell -Command *)",               // PowerShell
"Bash(cmd.exe /c *)",                        // cmd
"WebFetch",                                  // 无域名限制抓网页
"Bash(rm -rf node_modules)",                 // 删 node_modules
"Bash(rm -rf dist)",                         // 删构建产物
```

**建议**：先加上面 27 条跑 1 周，**频繁弹窗的**再追加。

## JSON 验证方法

加完后跑：

```bash
node -e "JSON.parse(require('fs').readFileSync('H:/AI-han/AiCode/.claude/settings.local.json','utf8')); console.log('✅ JSON 合法')"
```

不合法 = 漏了逗号/括号，**整个 settings 会失效**（CC 会用默认配置）。
