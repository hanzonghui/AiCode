## 📖 功能说明

### 1️⃣ 自动记忆（最常用）

**什么会自动触发？**

当对话中出现以下信息，AI 会自动调用记忆命令：

| 类型 | 触发词 | 示例 |
|:-----|:-------|:-----|
| 事实 | 日期、时间、地点、人物 | "年会12月25号" |
| 决策 | 决定用、选择了、方案 | "决定用Vue3" |
| 项目 | 项目名、技术栈、进度 | "项目A用MySQL" |
| 偏好 | 我喜欢、习惯用、总是 | "我喜欢用IDEA" |
| 人物 | 负责、对接、汇报 | "小王负责前端" |

**你不需要做任何操作，AI 会自动判断并记忆。**

### 2️⃣ 手动记忆

如果你想确保某条信息被记住：

```bash
# 方式1：直接在对话中说
"记住：公司年会在12月25号，国际会议中心3楼"

# 方式2：使用命令
bash ~/.claude/skills/left-brain/scripts/left-brain.sh remember "公司年会12月25号国际会议中心"
```

### 3️⃣ 搜索知识

```bash
# 精确搜索
bash ~/.claude/skills/left-brain/scripts/left-brain.sh recall "年会"

# 模糊搜索
bash ~/.claude/skills/left-brain/scripts/left-brain.sh recall "会议"

# 或者直接问 AI
"年会什么时候？"
"项目A用的什么技术？"
```

### 4️⃣ 列出所有知识

```bash
# 查看所有知识
bash ~/.claude/skills/left-brain/scripts/left-brain.sh list

# 分页查看（每页10条）
bash ~/.claude/skills/left-brain/scripts/left-brain.sh list 2
```

### 5️⃣ 监控面板

```bash
# 查看实时统计
bash ~/.claude/skills/left-brain/scripts/left-brain.sh dashboard
```

显示内容：
- 知识条目总数
- 分类分布（技术/人物/决策/事件）
- 最近访问的知识
- 系统状态

### 6️⃣ 系统状态

```bash
# 检查系统是否正常
bash ~/.claude/skills/left-brain/scripts/left-brain.sh status
```

## ❓ 常见问题

### Q: 知识会丢失吗？
A: 不会。知识存储在本地文件，只要不删除 `.claude/skills/left-brain/memory/` 目录就永久保存。

### Q: 知识太多怎么办？
A: 定期清理过期知识：
```bash
# 查看所有知识
bash ~/.claude/skills/left-brain/scripts/left-brain.sh list

# 手动删除不需要的知识文件
rm ~/.claude/skills/left-brain/memory/knowledge/KB-20260621-001.md
```

### Q: 换电脑怎么办？
A: 复制整个 `.claude/skills/left-brain/memory/` 目录到新电脑即可。

### Q: 搜索不到知识？
A: 检查知识文件是否存在：
```bash
ls -la ~/.claude/skills/left-brain/memory/knowledge/
```

### Q: 想修改已存储的知识？
A: 直接编辑知识文件：
```bash
# 用编辑器打开
notepad ~/.claude/skills/left-brain/memory/knowledge/KB-20260621-001.md
```
