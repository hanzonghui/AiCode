# 🧠 会话记忆功能说明

> 跨会话记忆 — 新会话自动加载上次对话摘要

---

## 🎯 功能说明

会话记忆功能让左脑能够：
1. **会话结束时** — 自动保存本次对话的摘要
2. **新会话开始时** — 自动加载上次对话的摘要
3. **跨会话延续** — 新对话可以无缝接续上次的内容

---

## 📝 使用方式

### 方式1：自动（推荐）

AI 会在以下时机自动执行：

**会话开始时：**
```bash
bash ~/.claude/skills/left-brain/scripts/session-init.sh
```

**会话结束前：**
```bash
bash ~/.claude/skills/left-brain/scripts/session-summary.sh save "摘要内容"
```

### 方式2：手动

```bash
# 保存摘要
bash ~/.claude/skills/left-brain/scripts/session-summary.sh save "本次讨论了xxx"

# 加载摘要
bash ~/.claude/skills/left-brain/scripts/session-summary.sh load

# 查看历史
bash ~/.claude/skills/left-brain/scripts/session-summary.sh list
```

│   - 读取 latest_summary.md                          │

```

┌─────────────────────┐     ┌─────────────────────┐
│ KB-20260621-001.md  │     │ session_20260621.md │
│ KB-20260621-002.md  │ ←→  │ session_20260622.md │
│ KB-20260621-003.md  │     │ session_20260623.md │
└─────────────────────┘     └─────────────────────┘
        ↑                           ↑
        │                           │
   长期知识存储                  短期会话记忆
   （永久保存）                 （按时间归档）
```

**分工：**
- **知识库** — 存储有价值的长期知识（技术栈、人物关系、决策）
- **会话记录** — 存储对话上下文（这次聊了什么、下次继续什么）
