## 🎯 记忆触发词

**事实**: 日期、时间、地点、人物、数字  
**决策**: 决定用、选择了、方案、确认  
**项目**: 项目名、技术栈、进度、模块  
**偏好**: 我喜欢、习惯用、总是、不要  
**人物**: 负责、对接、汇报、协作

## ❓ 快速排查

| 问题 | 解决 |
|:-----|:-----|
| 搜索不到 | `ls ~/.claude/skills/left-brain/memory/knowledge/` |
| 脚本无法执行 | `chmod +x ~/.claude/skills/left-brain/scripts/*.sh` |
| 知识丢失 | 检查 knowledge/ 目录是否存在 |
| 想清空 | `rm ~/.claude/skills/left-brain/memory/knowledge/*.md` |

---

## 🔄 每日流程

1. **开始对话** → AI 自动加载知识索引
2. **对话中** → 有价值信息自动记忆
3. **每5轮** → 自动显示监控面板
4. **结束对话** → 知识自动保存

---

## 💡 技巧

1. **说"记住"更可靠** → "记住：项目A用Vue3"
2. **直接问就行** → "项目A用的什么技术？"
3. **定期看监控** → 了解知识库增长
4. **清理过期知识** → 保持库精简
